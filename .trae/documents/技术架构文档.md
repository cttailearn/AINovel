# 技术架构文档

## 1. 项目架构概述

### 项目类型
前后端分离的 React 单页应用 (SPA)

### 技术栈
- **构建工具**: Vite 5.x
- **前端框架**: React 18.x
- **语言**: JavaScript / JSX
- **包管理**: npm
- **样式**: 原生 CSS

---

## 2. 项目结构

```
frontend/
├── config.json              # 后端配置文件
├── index.html               # HTML 入口文件
├── package.json             # npm 项目配置
├── vite.config.js           # Vite 构建配置
├── public/                  # 静态资源目录
│   └── config.json          # 构建时复制的配置文件
└── src/                     # 源代码目录
    ├── main.jsx             # React 应用入口
    ├── App.jsx              # 根组件
    ├── App.css              # 全局样式
    ├── config.js            # 配置读取工具模块
    └── components/          # 组件目录
        └── ConnectionStatus.jsx  # 连接状态组件
```

---

## 3. 核心模块设计

### 3.1 配置管理模块 (config.js)

**功能**：
- 读取 config.json 配置文件
- 提供后端 API 基础地址
- 管理配置状态

**API 设计**：
```javascript
// 获取后端完整 URL
export function getBackendUrl() {
  return `http://${host}:${port}`;
}

// 获取后端 API 地址
export function getApiBaseUrl() {
  return `${getBackendUrl()}/api`;
}
```

### 3.2 入口文件 (index.html)

**职责**：
- React 挂载点
- 页面标题和元数据

### 3.3 根组件 (App.jsx)

**职责**：
- 应用主容器
- 路由配置（可选扩展）
- 全局状态管理

### 3.4 连接状态组件 (ConnectionStatus.jsx)

**功能**：
- 显示后端连接状态
- 提供健康检查接口调用
- 展示连接信息

## 4. 配置管理

### 4.1 config.json 结构

```json
{
  "backend": {
    "host": "127.0.0.1",
    "port": 8008
  }
}
```

### 4.2 配置加载流程

1. **开发环境**：
   - Vite 读取 config.json
   - 通过 import() 动态导入
   - 注入到 window 对象供全局访问

2. **生产环境**：
   - 构建时复制 config.json 到 public 目录
   - 通过 fetch('/config.json') 加载配置

### 4.3 后端连接

- 默认地址：`http://127.0.0.1:8008`
- 健康检查接口：`/health_check` 或 `/health`
- API 前缀：`/api`

## 5. 脚本配置

### package.json scripts

```json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
```

### 运行说明

**开发模式**：
```bash
cd frontend
npm install
npm run dev
```
- 启动 Vite 开发服务器
- 默认端口：5173
- 支持热重载
- 自动读取 config.json 配置

**构建模式**：
```bash
cd frontend
npm run build
```
- 输出静态文件到 dist 目录
- 包含 HTML、CSS、JS 资源
- 复制 config.json 到输出目录

## 6. Vite 配置

### vite.config.js

```javascript
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { resolve } from 'path';
import fs from 'fs';

export default defineConfig({
  plugins: [react()],
  root: '.',
  build: {
    outDir: 'dist',
    assetsDir: 'assets',
  },
  server: {
    port: 5173,
    proxy: {
      '/api': {
        target: 'http://127.0.0.1:8008',
        changeOrigin: true,
      },
    },
  },
});
```

## 7. 后端接口对接

### 7.1 健康检查接口

假设后端提供健康检查端点：

**请求**：
```
GET /health_check
```

**响应示例**：
```json
{
  "status": "ok",
  "message": "Service is healthy"
}
```

### 7.2 API 请求封装

```javascript
export async function fetchHealthCheck() {
  const response = await fetch(`${getApiBaseUrl()}/health_check`);
  return response.json();
}
```
